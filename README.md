# Wing for Halo

基于 [Tokinx/Wing](https://github.com/tokinx/Wing) 移植、重构并适配 Halo 博客系统的主题。
原主题为 WordPress 版本，本项目已几乎完全重写前端结构与逻辑，保持风格一致。

## 使用资源说明

- 原主题：[Tokinx/Wing](https://github.com/tokinx/Wing)（GPLv2）
- 前端框架：Vue.js（MIT）
- CSS 框架：Spectre.css（MIT）
- 图标：草莓图标 **免费开源版**
- 其他工具库均为开源免费资源

## 安装

直接下载本项目放入 Halo 主题目录启用即可。
