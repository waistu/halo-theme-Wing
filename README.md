# ℹ️ 简介

[halo-theme-wring](https://github.com/waistu/halo-theme-wring) 基于 [Tokinx/Wing](https://github.com/tokinx/Wing) 移植、重构并适配 Halo 博客系统的主题。

原主题为 WordPress 版本，本项目已几乎完全重写前端结构与逻辑，保持风格一致。

## 📦 使用资源说明

- 原主题：[Tokinx/Wing](https://github.com/tokinx/Wing)（MIT）
- 图标：草莓图标 **免费开源版**

## 依赖资源

- CSS 框架：Normalize.css（本地）
- 字体图标：草莓图标 StrawberryIcon（本地自带 iconfont）
- 时间处理：dayjs + dayjs-plugin-relativeTime（本地）
- 代码高亮：Prism.js（CDN）
- 语言包：zh-cn 本地化（本地）

## 项目总结

本主题基于 **Thymeleaf** 模板引擎开发，使用原生 JavaScript（无框架）构建交互，依赖 Prism.js 代码高亮、dayjs 时间处理、草莓图标字体。
除 Prism.js 使用 CDN 外，其余资源均内置在主题中，无额外环境依赖。

## 安装使用

直接下载本项目放入 Halo 主题目录启用即可。
